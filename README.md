# FolioFriends 📈

국내·미국 주식 자산을 관리하고, 친구들과 포트폴리오를 **비중만 공유**하는 웹 앱입니다.
내 총 자산·종목별 수익을 확인하고, 금액은 가린 채 비중·수익률만 친구 피드/랭킹에 공유합니다.

**🔗 라이브 데모**: https://lee4jun2.github.io/foliofriends/

## 기능

- **자산 홈** — 총 자산, 오늘/총 손익, 자산 비중(스택바), 보유 종목 전체
  - 평가액 · 총수익 · 총수익률 · 일간수익 · 일간수익률 5가지 기준 오름/내림차순 정렬
  - 👁 눈 토글로 금액 숨기기 (비중·수익률은 유지)
- **종목 상세** — 가격 추이 스파크라인, 보유 현황, 평가 손익
- **친구 피드** — 친구들이 공유한 포트폴리오 (스택바 + 좋아요/댓글)
- **친구 포트폴리오** — 도넛 차트 + 비중 (금액 비공개, "나도 보유" 표시)
- **수익률 랭킹** — 친구들과 수익률 비교 (메달, 내 순위 강조)

> 상승=빨강 / 하락=파랑 — 국내 증시 관습을 따릅니다.

## 기술

- 순수 **HTML + CSS + Vanilla JS** — 빌드 과정 없음, GitHub Pages 정적 호스팅
- 차트(도넛·스택바·스파크라인)는 인라인 SVG로 직접 렌더
- 폰트: Pretendard (CDN)
- 외부 런타임 의존성 없음

## 로컬 실행

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

```
index.html   앱 셸 + 폰트/스타일 로드
```
